edunext-distro plugin for `Tutor <https://docs.tutor.overhang.io>`__
===================================================================================

Installation
------------

::

    pip install git+https://github.com/eduNEXT/tutor-contrib-edunext-distro

Usage
-----

::

    tutor plugins enable edunext-distro
